Инструкция по оплате

  • Оплата при помощи терминалов Easypay

    Наличными:

    Для оплаты найдите ближайший терминал Easypay. Карта терминалов
    Меню "Грошові перекази" -> "Поповнення гаманця, простір Easypay"

    Введите номер кошелька, учтите комиссию за оплату, 5 грн.
    Далее сохраните чек, желательно сфотографировать на экране, так как зачастую терминалы не выдают чек

    Смотреть видео как оплачивать


    Безналичный расчет (Visa, Mastercard)

    Перейти на страницу пополнения кошелька Easypay


    Сохраните чек после оплаты для подтверждения успешной оплаты на кошелек.

    Введите данные с чека, ID транзакции в забронированом заказе для произведения оплаты


  • Оплата EXMO, Биткоин (-4% стоимости)

    Выберите необходимый обменник с курсом Приват - Ехмо, либо Приват - Биткоин:

    Privat24 to Exmo

    Privat24 to Bitcoin


    Подробнее с работой сервиса можно ознакомиться здесь:

    Иструкция пользования сервисом bestchange



    Далее, следуете инструкциям обменника, в случае с ЕКСМО, вам на почту придет код, введя который на сайте вы совершите покупку.
    Либо, введете во время обмена адрес биткоин кошелька, который вам выдадут на сайте для зачисления суммы. Обычно это происходит в течении 30-60 минут.

    Внимание! Не пополяйте более одного раза биткоин кошелек, он становится недействительным. Всегда получайте новый адрес для оплаты



  • BTCU

    Для оплаты при помощи Btcu выберите в терминале IBOX пополнение BTCU, введите свой номер телефона, на него придет код, который вам будет необходимо ввести на сайте для использования кода


    (Внимание! При оплате данным методом, стоимость товара повышается на 20%, то есть если товар при оплате за биткоин стоит 100грн - то при оплате BTCU будет 124грн (20% комиссия сервиса BTCU и +4% так как при оплате биткином скидка))


    После пополнения, сохраните чек и введите необходимые данные на сайте для выполнения оплаты
